BLE Device Firmware

  • Category: Embedded C
  • Client: Climb Smart GmbH
  • Project date: 2019, 3 months
  • Technologies: Embedded C, BLE, I2C, Nordic Semiconductor SDK, Bluetooth LE, Oscilloscope, SWD, Git
  • Firmware development for the Zephyr RTOS based Nordic-Semiconductor SoC nRF52832
  • Bluetooth Low Energy based communication: Advertising, connection handling, error handling, services
  • Custom bluetooth services for broadcasting real-time sensor values
  • Development of a driver for the ADC nau7802
  • Over-the-air device firmware upgrade capabilities implemented
  • Programming and debugging via SWD
